# Neo4j Graph Explorer

A **Streamlit app** to visualize and explore Neo4j graph databases interactively.  
Run Cypher queries, inspect nodes and relationships, and explore your graph schema in real time.

---

## Features

- Run arbitrary **Cypher queries** from the sidebar.
- View **graph visualizations** using `streamlit-agraph`.
- Click nodes to see **properties and labels**.
- Browse **query history** and re-run previous queries.
- Explore **schema overview**: node types, relationship types, and top relationship patterns.
- Supports **priority-based node labeling** (`course_name` > `concentration` > `foundation` > first label).
